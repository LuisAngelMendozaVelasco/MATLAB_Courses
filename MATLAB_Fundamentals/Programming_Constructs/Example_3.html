Select country.
ctry = "Japan";

Price Analysis

Extract data for the chosen country
ctryPrices = prices(:, idx);
Make plot
plot(Year,ctryPrices, "o-")
grid minor;
xlabel("Year")
title(ctry + " Gas Prices")
