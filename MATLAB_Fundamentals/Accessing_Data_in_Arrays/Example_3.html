Electricity price data from January 1990 to December 2019 are stored in the matrix price. Each column is a year of monthly prices.

x = M(:,colNum);

What if you wanted to extract multiple rows or columns? You can index using a vector containing those index values.
For example, both lines of the following code will extract the first three rows of price.
firstThree = price(1:3,:);
firstThree = price([1 2 3],:);
quarters = price([1 4 7 10], :)
