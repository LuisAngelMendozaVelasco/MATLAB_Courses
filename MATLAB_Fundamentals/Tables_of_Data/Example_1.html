There are several ways to store text in MATLAB. Strings (created with double quotation marks) are the easiest to use in most cases.
To ensure text is imported as a string, set the TextType property to string.
data = readtable("dataFile.txt",...
"TextType","string")
EPL = readtable("./data/EPLresults.csv", "TextType", "string")
