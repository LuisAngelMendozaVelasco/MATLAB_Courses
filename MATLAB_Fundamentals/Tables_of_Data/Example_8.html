Task 1

You can add a new variable to a table using dot notation.
tableName.NewVarName = newData
totWins = EPL.HomeWins + EPL.AwayWins;
EPL.TotalWins = totWins;

Task 2

EPL.TotalDraws = EPL.HomeDraws + EPL.AwayDraws;

Task 3

To calculate a team's points, each win is worth three points and each draw worth one point.
EPL.Points = 3 * EPL.TotalWins + EPL.TotalDraws;
EPL = sortrows(EPL, "Points", "descend");

Task 4

EPL = removevars(EPL, ["HomeDraws" "HomeLosses" "HomeWins" "AwayDraws" "AwayLosses" "AwayWins"]);

Task 5

You can use the movevars function to move a column to a new location in a table.
data = movevars(data,"var2move","After",num)
Now var2move is the column after num.
EPL = movevars(EPL, "Points", "After", 1)
