How do you determine the number of values greater than 5 in each row of x? Recall that true and false are logical values equivalent to 1 and 0, respectively. If you perform any mathematical operation on a logical variable, it will automatically be converted to a numerical variable (with values of 0 or 1). This means you can use statistical functions on logical arrays. Summing logical values is equivalent to counting the true (1) values.
