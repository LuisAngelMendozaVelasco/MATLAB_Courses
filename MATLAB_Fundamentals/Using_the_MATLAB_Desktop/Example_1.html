Task 3

Notice the imported data looks a bit odd. For instance, the first four rows contain all NaNs, or "Not a Number". Remember, the data you are importing contains some header lines at the top of the file. Some of which contain empty cells. When the Import Tool encounters empty or unimportable cells, it will automatically place NaNs in those cells.
You can select only the desired part of the file by either setting the Selection range of the imported cells or by highlighting the desired cells with your mouse.
% This is the function you could use instead of the Import Tool.
gasprices2 = readtable("../data/gasprices.csv", "Range", "A5:K24");

The numeric data in the table looks correct, but the variable names for each column aren't quite right.
By default, the Import Tool will use the cells in the first row of the data file to define the variable names for each column. If the cell is empty, the variable will be named VarN where N is the column number.
% This is the function you could use instead of the Import Tool.
gasprices3 = readtable("./data/gasprices.csv", "NumHeaderLines", 3, "ReadVariableNames", true);
