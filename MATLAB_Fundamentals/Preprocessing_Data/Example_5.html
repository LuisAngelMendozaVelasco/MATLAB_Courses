Task 1

Each column of the matrix usage contains electricity usage for a given sector.
If you want to preserve the original data set, but ignore missing values for calculations like the average, you can use the "omitnan" flag. When you use the "omitnan" flag on the uncleaned data, the average is taken of each column, ignoring only individual entries in the calculation.
emean = mean(usage, "omitnan")
