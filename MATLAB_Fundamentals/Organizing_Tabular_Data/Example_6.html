Task 1

In the script, notice that the second variable name has some unusual underlines.
Whenever a column heading is not a valid MATLAB variable name, it is replaced by a valid name and the original column header is saved in the VariableDescriptions property of the table. VariableDescriptions contains a cell array of text – one cell for each column header.
orig = teamInfo.Properties.VariableNames{2}
orig = 'Payroll_M__'

Task 2

You can see in orig that the characters for parentheses and £ were modified to underscores.
teamInfo.Properties.VariableNames{2} = 'Payroll'
