Task 1

The variable ob contains a list of objectives for fifteen crash tests.
Many objectives are related to getting passenger restraint information. To find all of the test objectives that contain the word "restraint", use the contains function.
newstr = contains(str,pattern,"IgnoreCase",true)
This returns a logical array that identifies which elements of str contain pattern regardless of capitalization.
idx = contains(ob,"restraint","IgnoreCase",true)
idx = 15×2 logical array
0 1 0 0 0 1 0 0 0 1 0 1 0 1 0 0 0 1 0 1
restraint = ob(idx)

Task 2

You can see that the vector ob has extra text that does not add much and in fact clutters the data. For instance, you do not need each test objective to start with the word "objective."
The erase function allows you to erase certain expressions from a string array.
newstr = erase(str,pattern)
resClean = erase(restraint,["OBJECTIVE:" "TO OBTAIN"])
resClean = 11×1 string
