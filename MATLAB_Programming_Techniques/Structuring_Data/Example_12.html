Task 1

In this activity, you will collect test data about the Ford Expedition in a single structure. First, you will start by creating the structure using dot notation.
crashData.car = '2018 Ford Expedition';

Task 2

Now that you have created your structure, you can add a field containing the weight (lbs), length (mm), and width (mm).
Note that fields do not need to be the same data type.
crashData.carSize = [2619 5348 2031];

Task 3

Several variables about the Ford Expedition crash test are defined in the live script. In this task, you will finish storing them together in a structure.
