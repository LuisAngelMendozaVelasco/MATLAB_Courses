Incorporating Tabular and Non-tabular Data

Task 1

In this exercise, you will set the data type of text variables in a table and convert it to a structure array so you can add non-tabular data.
By default, readtable imports text data as character vectors. Any text extracted from the table is returned as a cell array of character vectors. You can see this default behavior demonstrated at the top of the open live script.
You can specify the way readtable imports text by setting the "TextType" option to "string" or "char".
fleet = readtable("./data/fleet.csv", "TextType", "string")

Task 2

The type and testinfo columns are labels that split the data into categories according to car type and the type of test performed on the car.
The convertvars function allows you to convert several variables at once with the syntax:
convertvars(table,["Var1" "Var2"],"datatype")
fleet = convertvars(fleet, ["type" "testinfo"], "categorical")
