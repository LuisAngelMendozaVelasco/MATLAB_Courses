Task 2

Cell and structure arrays can contain non-tabular data, but you may need to work only with a portion of the data that is tabular.
The structure crashData has a field called data, which contains a cell array of differently sized sensor data. If you don't need the sensor data, you can remove it and work with the data as a table.
The rmfield function removes fields from a structure with the following syntax:
rmfield(structure,fields)
testStruct = rmfield(crashData, "data")
