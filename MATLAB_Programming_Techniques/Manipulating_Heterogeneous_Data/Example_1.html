S.fieldName

Task 4

As with extracting a field from each element of a structure, extracting multiple elements of a cell array returns a comma-separated list, which again looks like a list of assignments to ans.
When a command returns a comma-separated list as output, you can assign the elements of that list to multiple variables using square brackets, [ ].
[x1,x2] = s(1:2).fieldName
The syntax is similar how you request multiple outputs from a function.
[ct, cx] = c{1:2}
