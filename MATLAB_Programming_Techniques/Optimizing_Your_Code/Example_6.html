Preallocation Experiment

The local functions below create an matrix A for which the element is equal to . The only difference is that one has preallocated memory for A and the other has not.
This code is used to test the function. Do not edit.

The local function in preallocateMakeA.mlx creates an m-by-n marix A, but memory is not preallocated for A. You can run the script a few times to get a feel for the time required to calculate A.
function A = makeA(m,n)
 
% preallocate A
A = zeros(m,n);
for I = 1:m
for J = 1:n
A(I,J) = I+J;
end
end
 
end
