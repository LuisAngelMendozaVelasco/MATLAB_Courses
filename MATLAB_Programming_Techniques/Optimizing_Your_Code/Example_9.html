Nesting a Function

The following function finds the roots of the polynomial near n.
a = 1;
b = 0;
c = -4;
n = 1;
 
z = findZero(a, b, c, n)
z = 2
The function findZero takes in the coefficients of a polynomial as inputs and finds the root of the polynomial nearest to the specified value.
The command z = findZero(1,0,-2,1) returns an error because the function polynomial is a local function and does not have access to the variables a, b, and c.
