Loading Data into a Structure

The following code creates a structure of crash test images.

Tasks 1 & 2

Recall that to preallocate a structure of length n, you need to assign a structure with the appropriate fields to the nth element.
S(n) = struct('field1',6,'field2',7)
folders = dir("./data/photos");
folders = string({folders.name});
folders = folders(~startsWith(folders, "."))
folders = 1×6 string
"Test1" "Test2" "Test3" "Test4" "Test5" "Test6"
 
% Preallocate the structure crashImages
crashImages(length(folders)) = struct("Name", folders(end), "Images", cell(1));
 
for k = 1:length(folders)
photos = dir("./data/photos" + filesep + folders(k));
photos = string({photos.name});
photos = photos(~startsWith(photos,'.'));
% Preallocate the cell array images
images = cell(length(photos),1);
for kk = 1:length(photos)
im = imread("./data/photos" + filesep + folders(k) + filesep + photos(kk));
images{kk} = im;
end
crashImages(k).Name = folders(k);
crashImages(k).Images = images;
clear images
end
