Get File ID

filename is the name of a file.
fileID is the file id number.
The function fopen opens a file and returns an integer called the file identifier. If the file cannot be opened, fopen returns -1
function fileID = processFile(filename)
warning("on","all")
% Try to open the data file
fileID = fopen(filename);
