ODE function
Do not edit. This code defines the ODE function for the system of ODEs
function dYdt = odefun(t, Y)
x = Y(1); % prey
y = Y(2); % predator
dxdt = x - 1/2*x*y;
dydt = -3/4*y + 1/4*x*y;
dYdt = [dxdt; dydt];
end
