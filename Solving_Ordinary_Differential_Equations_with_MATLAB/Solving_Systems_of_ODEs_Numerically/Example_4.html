tRange = [0 3];

ODE function
Do not edit. This code defines the ODE function for the system of ODEs
function dYdt = reactionODE(t, Y)
x = Y(1); % chemical x
y = Y(2); % chemical y
z = Y(3); % chemical z
dxdt = -2*x;
dydt = 2*x - 0.02*y^2;
dzdt = 0.01*y^2;
dYdt = [dxdt; dydt; dzdt];
end
