ODE function
Do not edit. This code defines the ODE function for the system of ODEs
function dYdt = reactionODE(t, Y)
x = Y(1); % chemical x
y = Y(2); % chemical y
dxdt = -2*x;
dydt = 2*x;
dYdt = [dxdt; dydt];
end
