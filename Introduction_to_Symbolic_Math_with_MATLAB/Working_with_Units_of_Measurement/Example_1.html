Task 1

Use units when your variables or values represent real life measurements.
To assign units, you first need to create a variable that contains a units collection. A units collectoin is the set of all units available for you to assign in MATLAB. Click here for a complete list of all units.
The command
u = symunit
uses symunit to store a units collection in the variable u.
