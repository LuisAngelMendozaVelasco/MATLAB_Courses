Task 1

The subs function takes an expression and replaces all instances of a symbolic variable with a value.
v = subs(expression,variable,value)
For example, the following code takes the expression z and replaces the symbolic variable a with the value 4:
syms a
z = 3*a^2;
out = subs(z,a,4)
out =
48
The result is 3*42 = 48.
