Task 1

The Pythagorean theorem can be written as an equation relating the lengths of the sides a, b, and c of a right triangle.
a^2+b^2=c^2
In this activity, you'll explore values for a and b that solve this equation when c is 13.
You can use assume to apply assumptions to multiple variables at once. For example, to assume x, y and z are all integers, use the command
assume([x y z],"integer")
assume([a b], "integer")

Since a and b are lengths of sides of a triangle, it would not make sense for them to be negative or zero.
If you use the assume function on a variable again, it will remove previous assumptions on that variable and replace them with the new assumption. To make additional assumptions on a variable, use the assumeAlso function.
assumeAlso(var,"positive")
assumeAlso([a b], "positive")
