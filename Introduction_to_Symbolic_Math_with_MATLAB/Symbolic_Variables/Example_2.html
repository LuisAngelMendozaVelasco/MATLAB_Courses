Symbolic Variables

Instructions are in the task pane to the left. Complete and submit each task one at a time.

Task 1

Like in traditional equations, symbolic variables don't need to contain an assigned value.
You can use the syms command followed by the variable name to create a symbolic variable without assigning it a value.
syms y
The syntax above creates a symbolic variable y with no assigned value.
syms x

Task 2

You can also use syms to create multiple symbolic variables at once.
The following command creates the symbolic variables x, y, and z using just one line of code.
syms x y z
syms a b c
