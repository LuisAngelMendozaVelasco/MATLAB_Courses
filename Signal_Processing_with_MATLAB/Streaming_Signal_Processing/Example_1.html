Task 1

To stream a signal with the DSP System Toolbox, you can generate a signal or import one from the Workspace with:
dsp.SignalSource(sig,samp)
The second input samp is the number of samples per frame that you want to import. A frame is one portion of the signal.
src = dsp.SignalSource(monitoredfan,1000)
src =
dsp.SignalSource with properties: Signal: [1200000×1 double] SamplesPerFrame: 1000 SignalEndAction: 'Set to zero'

Task 2

Now that you've set up your System object, the next step is to turn it on.
To stream one frame of your signal, you can use the object that you just created with empty parentheses (()):
s = dsp.SignalSource(sig,samp)
frame1 = s()
frame2 = s()
