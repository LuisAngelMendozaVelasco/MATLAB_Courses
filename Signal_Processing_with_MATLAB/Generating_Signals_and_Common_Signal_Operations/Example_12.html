Create Square Wave

Instructions are in the task pane to the left. Complete and submit each task one at a time.

Task 1

Any periodic signal can be represented by adding sine waves together. This is key to spectral analysis, which is the focus of the next two modules. To demonstrate this concept, you'll make a square wave.
fs = 100;
t = (0:1/fs:3)';

Task 2

Square waves are sums of odd harmonics where the amplitude of each harmonic is the reciprocal of the harmonic number.
y=1/Asin(2πAt)
y1 = 1*sin(2*pi*1*t)
y1 = 301×1
0 0.0628 0.1253 0.1874 0.2487 0.3090 0.3681 0.4258 0.4818 0.5358

Task 3

The amplitude for the 3rd harmonic should be 13, using the same formula:
y=1/Asin(2πAt)
y3 = 1/3*sin(2*pi*3*t)
y3 = 301×1
0 0.0625 0.1227 0.1786 0.2282 0.2697 0.3016 0.3229 0.3327 0.3307

Task 4

plot(t, y1+y3)
grid on

Task 5

With just two harmonics added together, you can start to see a square wave. More harmonics will make the square wave more accurate.
y5 = 1/5*sin(2*pi*5*t)
y5 = 301×1
