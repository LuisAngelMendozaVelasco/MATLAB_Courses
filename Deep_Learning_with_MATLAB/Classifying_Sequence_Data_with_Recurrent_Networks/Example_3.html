Task 1

Locate the prepsig function at the bottom of the script. The function contains cell2mat, which you used in the previous activities to visualize one observation. How do you process the entire data set?
You can transform your signal datastore using a transformation function.
dst = transform(ds,@fcn)
The second input is a function handle, which is how you pass a function to another function. To create a handle for your transformation function, precede the function name with an @ sign.
sigt = transform(sigds, @prepsig)

Tasks 3 & 4

All the observations were imported into one big matrix. How can you tell the observations apart? While training an LSTM, the network would think this is one observation with over 20,000 channels.
To fix this, you can store the processed signal in a cell.
sigout = {sigout};
Look at the size of the cells in sigdata. Each cell contains a matrix with three columns, which correspond to the acceleration in the x, y, and z direction.
The channels are stored in columns, but to use this data to train a network, they need to be stored in rows. You can fix this by transposing the matrix with the transpose operator (').
row = col'
function sig = prepsig(sigin)
sig = cell2mat(sigin);
sig = sig';
sig = {sig};
end
