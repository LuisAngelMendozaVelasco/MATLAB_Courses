Task 1

Validation data can be a datastore, table, or cell array. Because the images and labels for this data set are stored in two different variables, it is simplest to create a cell array.
A cell array of validation data should have two elements, where the first contains the images and the second contains the labels.
validationData = {X Y}
valData = {XVal YVal};

Task 2

You can set the "ValidationData" option to turn on validation.
Because this network trains very quickly, you can also decrease the validation frequency so that the network is validated more often.
