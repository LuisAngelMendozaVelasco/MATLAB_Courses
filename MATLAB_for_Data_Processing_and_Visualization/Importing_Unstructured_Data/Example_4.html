Task 1

You can read data into MATLAB using the textscan function with two required inputs: a file identifier and a format specification string.
data = textscan(fid,"%D%q%f%f%f")
The above code will attempt to convert the first unit of data to a date, "%D", the second to a string, "%q", and the next three units to double precision numbers, "%f". This pattern is repeated indefinitely, so the sixth unit is a date, the seventh is a string and so on.
For formatted text files, typically the format string will provide a specifier for each column of data.
fid = fopen("./data/file1.txt")
fid = 6
data = textscan(fid,"%f%f")

The file identifier and the format specification string are the only required inputs to textscan, but unstructured files often require other settings to import data correctly.
