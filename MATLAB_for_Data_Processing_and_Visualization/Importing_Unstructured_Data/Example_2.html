Task 1

The file hurricaneData1960.txt is open in the editor so you can familiarize yourself with the data's structure. The first line of the file contains information about the block of data that follows.
To parse this file, you'll first need to extract information from the two headers.

Task 2

The character vector info contains the following:
  1. ANNA: Name of the hurricane
  2. M = 7: Number of data points for this hurricane
  3. SNBR = 902: Station number
You can use text processing functions to extract information from info. To get the number following M, you can use extractBetween.

Task 3

M is a cell containing a string, but a numeric value makes more sense.
There is tabbed white space around the 7, which is indicated by the '->' symbol included in the output. The str2double function will automatically ignore that.
M = str2double(M)
M = 20

Task 4

If your desired string is at the end of your text, you can use the extractAfter function.
SNBR = extractAfter(info, "SNBR =")
SNBR = ' 902 '
SNBR = str2double(SNBR)
SNBR = 902
