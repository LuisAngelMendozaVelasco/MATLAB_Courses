File I/O Basics

Instructions are in the task pane to the left. Complete and submit each task one at a time.

Task 1

You can use the fopen function to open a file. This does not open the file in an editor but instead opens a connection between MATLAB and the file for reading and writing the data.
fi = fopen("economy.txt");
fi =
3
The returned value is a unique file identifier used to reference the open file.
fid = fopen("./data/foo.txt")
fid = 3

Task 2

Once the file is opened, you can use the file identifier to read data from the file.
fgetl(fi)
ans =
date, unrate, gdp, feddebt
fgetl(fi)
ans =
2014-01-01, 6.6, 15724.9, 103.63233
x = fgetl(fid)
x = '05-Jun-2001 12 27.5 95.0 46 1007'

Task 3

Subsequent file reads will read in subsequent lines. The file position indicator moves to the beginning of the next line after each read.
x2 = fgetl(fid)
x2 = '05-Jun-2001 18 28.5 95.3 58 1002'

Task 4

You can move the file position indicator back to the beginning of the file using the frewind command.
frewind(fi)
frewind(fid)
y = fgetl(fid)
y = '05-Jun-2001 12 27.5 95.0 46 1007'

Task 5

You've seen how fopen creates a link to a file. This requires resources from your operating system. If you open files without closing them when you're finished you are wasting system resources and risk opening more files than is allowed by your system. This will result in a MATLAB error.
To close a file that has been opened by fopen, use the fclose function.
fclose(fi)
fclose(fid)
