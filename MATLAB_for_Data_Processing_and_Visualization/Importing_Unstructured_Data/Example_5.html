Task 2

The third element in elements is 'Period'. That's not supposed to be in your data! It's another header that separates two blocks of data.
Instead of importing the entire column, you can import just the first block by specifying n lines.
x = textscan(fid,"fmt",n)
If you make a mistake, you can rewind back to the start of your file by passing the file identifier to the frewind function.
