Task 1

Measurements were taken six hours apart, so an approximate duration can be obtained by counting the number of measurements for each hurricane.
If you pass a table and a grouping variable to groupsummary, the output will be a table with the group names and a count of how many observations are in each group.
grCounts = groupsummary(tbl,"GrpVar")
Note that GrpVar must be a column of tbl.
numReadings = groupsummary(data, "ID")

Task 2

If you want to perform a numeric calculation, you should also specify the function and variables that you want to be calculated for each group as the third and fourth inputs.
grStats = groupsummary(tbl,"GrpVar",...
"method","DataVar")
You can pass many statistical functions, like "max", as a string. Or you can pass a function handle.
List of methods to pass to groupsummary
You will notice that the first two columns still contain the group names and counts.
statsHurrs = groupsummary(data, "ID", "max", "Windspeed")

Task 4

You can also specify multiple statistics. Each statistic will be calculated for every variable you specify.
Each variable you specify must be compatible with every statistic. So if you specify ["max" "mean"], you can no longer use the "HurrCat" variable. While there is a maximum of an ordinal category, the average is not defined.
statsHurrs3 = groupsummary(data, "ID", ["min" "range" "mean"], ["Timestamp" "Windspeed"])

Task 5

The table produced by groupsummary may benefit from some post-processing.
For instance, if your primary goal was to calculate the first timestamp, duration, and the average wind speed of each hurricane, you may not need the other variables. You may also want to rename variables so they are shorter and more descriptive.
You can use the removevars and renamevars functions to remove and rename variables. For each function, specify variable names as as string vector.
newTbl = removevars(tbl,vars2rm)
newTbl = renamevars(tbl,oldVarNames,newNames)
statsHurrs3 = removevars(statsHurrs3, ["GroupCount" "mean_Timestamp" "min_Windspeed" "range_Windspeed"])
