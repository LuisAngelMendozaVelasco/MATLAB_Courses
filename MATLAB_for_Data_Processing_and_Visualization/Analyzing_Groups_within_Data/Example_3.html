Task 1

If you want to calculate statistics over land and sea for each hurricane, you need to group by both the hurricane ID and the Location variables. To do this, you can specify them in a vector of strings.
stats = groupsummary(tbl,...
["GrpVar1" "GrpVar2"])
The output will have the first grouping variable as the first column and the second in the second column. The third column contains the number of observations in that group
overLandAndSea = groupsummary(data, ["ID" "Location"])
