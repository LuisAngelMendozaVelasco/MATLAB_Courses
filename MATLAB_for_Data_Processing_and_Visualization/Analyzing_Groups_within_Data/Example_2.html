Task 1

You can use a datetime variable as the third input to specify a time period as the group.
stats = groupsummary(tbl,"DatetimeVar",...
"groupbin")
The groupbin input let's you group data by second, by century, and many time intervals in between. The output is a table with the group count for each time period.
See the documentation for a list of possible datetime group bins.
The variable statsHurrs contains statistics about each hurricane, including the timestamp for the first measurement, which is contained in the variable FirstTS.
byDecade = groupsummary(statsHurrs, "FirstTS", "decade")

Task 2

You can also specify a method for each time period and the variables to which to apply it.
stats = groupsummary(tbl,"DatetimeVar",...
"groupbin",...
"method","DataVar")
As before, you must make sure the specified method can be applied to all of the data variables.
wsDecade = groupsummary(statsHurrs, "FirstTS", "decade", "mean", "max_Windspeed")
