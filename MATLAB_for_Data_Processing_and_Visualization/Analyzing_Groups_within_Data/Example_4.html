Task 2

Use the unstack function to create a new table from data in a specified column.
newTbl = unstack(tbl,"DataVar","ColVar")
The variable newTbl will have column names coming from the ColVar variable of the original table. The table elements come from the values in DataVar and are organized in rows by the remaining grouping variable.
hurrCats = unstack(catsByHurr, "GroupCount", "HurrCat")
