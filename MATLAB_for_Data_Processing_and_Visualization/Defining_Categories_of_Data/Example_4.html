Infinite Ranges

Instructions are in the task pane to the left. Complete and submit each task one at a time.
This code sets up the activity.
binedges = [0 0.2 0.6 0.9];
x = [ 0.90 0.32 0.95 NaN 0.87 0.71 0.42 0.60];

Task 1

In this activity, you will see how discretization works on edge cases. Notice that x has missing data.
y = discretize(x, binedges)
y = 1×8
3 2 NaN NaN 3 3 2 3

Task 2

Any NaNs are unclassified in y, or, left as NaNs. The third element of x isn't missing, but it was left unclassified too. That is because it is outside of the bin edges defined in binedges.
If you want to include bins for values outside of the edges, add -Inf or Inf to the vector of bin edges.
binedges = [0 0.2 0.6 0.9 Inf]
binedges = 1×5
0 0.2000 0.6000 0.9000 Inf

Task 3
