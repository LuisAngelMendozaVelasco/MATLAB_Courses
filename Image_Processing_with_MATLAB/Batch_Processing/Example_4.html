Task 1 & 2

To process the roundworm images, you first need to locate the image files. With an image datastore, you can define their location and only import them when you're ready to process them.
When you create an image datastore, provide it with the directory you want to search and it will find the images in that folder. If you also want to search subfolders, set the "IncludeSubfolders" option to true.
ds = imageDatastore("directory",...
"IncludeSubfolders",true)
The data folder names categorize the roundworm images; the "alive" folder contains the images of live worms while the "dead" folder contains images of dead worms. This information is an important part of the workflow: when selecting a cutoff for the median worm length you need to differentiate between lengths for live and dead worms.
You can create labels from the folder names by setting the "LabelSource" option to "foldernames" when creating the datastore.
ds = imageDatastore(...
"LabelSource","foldernames")
ds = imageDatastore("./images/antibioticResponse",...
"IncludeSubfolders", true,...
"LabelSource", "foldernames")
