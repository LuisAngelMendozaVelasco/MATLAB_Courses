Increasing Contrast

Instructions are in the task pane to the left. Complete and submit each task one at a time.
This code sets up the activity.
roundworms = imread("./images/worms.jpg");
imshow(roundworms)

Task 1

Sometimes, an image's intensities have a very small range. The image worms.jpg is of a medical slide from an experiment testing the effect of antibiotics on roundworms. Notice that you can't see anything because all of the intensities are about the sameall very low.
You can use the imhist function to display a histogram of an pixel intensities of an image.
imhist(I)
imhist(roundworms)

Task 2

The imadjust function automatically adjusts intensity values to increase contrast.
Iadj = imadjust(I);
It adjusts the values to fill the entire available range by saturating the top and bottom 1% of intensity values.
rwAdj = imadjust(roundworms);
imshow(rwAdj)

Task 3

imhist(rwAdj)
