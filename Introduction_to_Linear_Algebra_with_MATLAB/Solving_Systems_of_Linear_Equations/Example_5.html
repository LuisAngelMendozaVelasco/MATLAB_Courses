Reorganize and Solve a Linear System

Instructions are in the task pane to the left. Complete and submit each task one at a time.

Task 1

Larger systems of equations don't fit neatly on a plot. However, the same mathematical techniques apply.
Consider the following system of linear equations:
p+2q−3r+4s=12
3s−2r+2p=10−2q
q+r+1=0
p−2s−q+r=−4
Before you put this system of equations into matrix form, you need to rearrange the equations.
A = [1 2 -3 4; 2 2 -2 3; 0 1 1 0; 1 -1 1 -2];

Task 2

The equations should be reorganized to
1p+2q−3r+4s=12
2p+2q−2r+3s=10
0p+1q+1r+0s=−1
1p−1q+1r−2s=−4
b = [12; 10; -1; -4];

Task 3

The equations are now assembled into a single matrix equation:
or more simply
Ax=b
You can solve for x using the backslash operator (\).
x = A\b
x = 4×1
1 0 -1 2
