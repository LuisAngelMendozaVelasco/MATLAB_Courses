Task 1

Notice that there are only four singular values in the matrix S. This means that the last three columns of U are unused and unnecessary for reconstructing X.
You can use the "econ" option to compute the economy size SVD, which returns only the essential information for reconstructing a matrix.
[U,S,V] = svd(A,"econ")
[u, s, v] = svd(X, "econ")
